# GOJ
My Online Judge:
http://114.55.106.99/

![](screenshot/home.png)


GOJ is an open source OJ inspired by Vmatrix. It use Vue as front-end, SpringBoot as back-end, run python as judger in docker to increases security and resource-friendly.

# Dependency
- Vue, Vuetify
- SpringBoot
- Python
- Redis
- Mysql5.7
- Docker
